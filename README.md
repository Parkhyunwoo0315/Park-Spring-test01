# Park Spring Test 01

Spring Boot 학습을 위한 테스트 프로젝트입니다.

## 개발 환경

* Java 17
* Spring Boot
* Gradle
* IntelliJ IDEA

## 실행 방법

터미널에서 아래 명령어를 실행합니다.

./gradlew bootRun

## 프로젝트 구조

src/main/java
src/main/resources
src/test/java

## 학습 목적

* Spring Boot 프로젝트 구조 이해
* Gradle 기반 빌드 방식 학습
* Java와 Spring Boot 기본 실행 흐름 학습
